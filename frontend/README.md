# Frontend

## Stack
- React 18
- TypeScript
- Vite 6
- Tailwind CSS 4

## Local run
```bash
cp .env.example .env
npm install
npm run dev
```

Default app URL: `http://localhost:5173`

Set API base URL with `VITE_API_BASE_URL` in `.env`

## Quality checks
```bash
npm run lint
npm run typecheck
npm run build
```
