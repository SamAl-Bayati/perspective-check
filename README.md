# PerspectiveCheck

PerspectiveCheck is a web app for projecting 3D model geometry onto a 2D canvas.

Projection model used by the app:
- `x' = x / z`
- `y' = y / z`

Target 3D file support:
- OBJ
- glTF and GLB
- STL
- 3MF
- FBX
